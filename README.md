# Gaming-Controller-Housing-BOTTOM-SHELL
-A python coded 3D model using build123d and opencasecad to create a featuring precise fitment, ergonomic curves, and print-ready geometry. Designed to house standard controller internals with easy assembly.

- Volume difference checker is also mentioned to validated the precisenes of model.
